🌾 AgriSetu – Smart Agriculture Assistant

AgriSetu is an AI-powered web application designed to help farmers make informed decisions by providing crop price predictions, weather insights, and smart recommendations. The platform bridges the gap between farmers and technology to improve productivity and profitability.

🚀 Features
📊 Crop Price Prediction
Predict future market prices of crops using machine learning models
🌦️ Weather Insights
Real-time weather updates for better farming decisions
📍 Location-Based Data
State and district-level agricultural insights
📈 Interactive Dashboard
Visual charts and statistics for better understanding
🤖 AI Recommendations
Smart suggestions for crop planning and selling

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/agrisetu.git
cd agrisetu
2️⃣ Setup Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
3️⃣ Setup Frontend
cd frontend
npm install
npm run dev
🌐 Usage
Open the frontend in your browser
Select your state, district, and crop
View:
Predicted prices 📊
Weather data 🌦️
Insights & recommendations 🤖
🧠 How It Works
Historical agricultural data is processed using ML models
Models predict crop prices based on trends
Weather APIs provide real-time climate data
Frontend displays results in an interactive dashboard
